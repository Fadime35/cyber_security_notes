## DNS (Domain Name System) 

DNS, internet üzerindeki alan adlarını (ör. www.google.com), bilgisayarların anlayabileceği IP adreslerine (ör. 142.250.190.78) çeviren sistemdir.

**Amaç ve Kullanım:**

- İnsanlar web sitelerine isimlerle erişmek ister (www.example.com) ama bilgisayarlar ve sunucular IP adreslerini kullanır.

- DNS, bu çeviriyi yaparak kullanıcıların siteye kolayca ulaşmasını sağlar.

- Böylece karmaşık sayılardan oluşan IP adreslerini ezberlemeye gerek kalmaz.

**Örnek:**

- Tarayıcıya www.google.com yazılır.

- DNS sunucusu bunu 142.250.190.78 IP adresine çevirir.

- Tarayıcı bu IP adresine bağlanarak Google sayfasını açar.


DNS, internetin rehberi gibidir: Alan adını bulur, doğru IP’ye yönlendirir, web sitelerine kolay erişim sağlar.
